
     ======================================================================
          対話知能学 旅行代理店タスク対話コーパス（Tabidachi） 説明書     
     ======================================================================

                                                                 2024.08.20
                                                               電気通信大学
                                                                  稲葉 通将


本コーパスは、文部科学省 科学研究費助成事業「新学術領域研究（研究領域提案型）」の
「人間機械共生社会を目指した対話知能システム学（略称：対話知能学、8105）」において
収集された、人同士の対話映像データとそれに対するアノテーションデータで構成される。

旅行代理店の対話システムを想定し、旅行代理店のスタッフ役（以下「オペレーター」）
とカスタマー役（以下「ユーザー」）の観光プランに関する相談の対話を、Zoomを介して
収録したもので、オペレーターは観光情報検索用のシステムを操作しながら対話を行い、
ユーザーは所定のシナリオに基づきオペレーターとの対話により旅行プランを決定する。

話者は、人材派遣会社を通して募集しており、オペレーター役は旅行代理店に勤務経験が
ある成年5名、ユーザー役は成年25名・高齢者10名・こども20名の男女計55名である。
対話時間は1回20分程度、対話の総数は330（1ユーザーあたり6回）である。


【データ収集手順】

① オペレーター役5名、ユーザー役55名の実験参加者を手配した。
② オペレーター役は検索システムを使用して、ユーザー役に対し観光地の案内を行った。
③ ②の対話をZoom上で行い、その様子を録画・録音した。
④ オペレーター役が使用した検索システムの挙動を取得するため、PCの画面をキャプチャ
   録画した。
⑤ ユーザー役参加者は旅行先に関する設定を収録前後にアンケート入力した。


【収集結果】

オペレーター役とユーザー役は、Zoom上で、対面方式で3回、検索システムを画面共有する
方式で3回、計6回の対話を行った。（オペレーター役は6回を通して同一）

各対話は、事前に旅行先、出発する季節、人数、2人以上の場合のメンバー構成をアンケート
取得した「通常シナリオ」と、「どういう旅行がしたいか」を事前に20文字程度でアンケー
ト収集した「ゆるいシナリオ」の2種類のシナリオを使用した。
シナリオ実施配分と対話方式の組み合わせ、実施の順番は以下の通り。

  対話1：対面による対話 / 通常シナリオ
  対話2：対面による対話 / 通常シナリオ
  対話3：対面による対話 / ゆるいシナリオ
  対話4：画面共有による対話 / 通常シナリオ
  対話5：画面共有による対話 / 通常シナリオ
  対話6：画面共有による対話 / ゆるいシナリオ


【収集期間】

2020年11月10日 ～ 2021年2月25日


【ファイル命名規則】

フォルダ名、ファイル名内の各IDの内訳は以下の通り。

  オペレーターID：1～5
  ユーザーID：101～125（成年25名）、201～210（高齢者10名）、301～320（こども20名）
  対話ID：1～6

フォルダの命名規則は以下の通り。

  [ユーザーID]_[対話ID]_[オペレーターID]


【提供データ】

各対話の映像、音声、画面キャプチャ、発話の書き起こしは以下のファイルに含まれる。
（「対話知能学」のプロジェクト内で2021年9月に共有したもの）

  Tabidachi2109-1.zip ：ユーザーID 101～125 の成年25名分
  Tabidachi2109-2.zip ：ユーザーID 201～210 の高齢者10名分
  Tabidachi2109-3.zip ：ユーザーID 301～320 のこども20名分

zip内の各フォルダには以下のデータを収録している。

  [ユーザーID]_[対話ID]_[オペレーターID]_operator.m4a：オペレーターのみの音声
  [ユーザーID]_[対話ID]_[オペレーターID]_user.m4a    ：ユーザーのみの音声
  [ユーザーID]_[対話ID]_[オペレーターID]_zoom.wav    ：オペレーターとユーザーの音声
  [ユーザーID]_[対話ID]_[オペレーターID]_zoom.mp4    ：zoom上で録画した動画
  [ユーザーID]_[対話ID]_[オペレーターID]_capture.mp4 ：オペレーターの操作画面をキャプチャした動画
  [ユーザーID]_[対話ID]_[オペレーターID]_zoom.tsv    ：音声の書き起こし（記号<>は聞き取り不可）


Tabidachi-info.zip には以下のファイルを収録している。

  audio_info.json ：各音声ファイルの長さ，サンプリング周波数，チャンネル数
  movie_info.json ：各動画ファイルの長さ，FPS，縦横の大きさ，総フレーム数
  operator.json   ：オペレーターに関する情報
  user.json       ：ユーザーに関する情報
  questionair.json：ユーザーから収集したアンケート
                   （事前に設定したシナリオと、対話の結果決まった行先など）


Tabidachi-annotation.zip には以下のファイルを収録している。

  annotations/  ：各対話のアノテーションデータ（jsonファイル）
  tag_list.pdf  ：アノテーションに用いたタグのリスト
  spot_info.json：対話中に観光情報検索用システムより取得した観光地情報（※注）

（※注）
本対話収録では観光情報の検索に「るるぶDATA API」を用いています。
この観光地情報は「るるぶDATAサービス利用規約」に準じた取り扱いをお願いします。

  119_2_5_zoom.mp4
  119_3_5_zoom.mp4
  123_1_4_zoom.mp4
  123_2_4_zoom.mp4
  123_3_4_zoom.mp4

以下の動画では、途中で画面上の位置が変更されている。

  208_1_2_zoom.mp4

そのため、これらの動画と同じフォルダに、編集により画面上の位置を修正したファイル
を配置している。ファイル名は以下の通り。

  119_1_5_zoom_flip.mp4
  119_2_5_zoom_flip.mp4
  119_3_5_zoom_flip.mp4
  123_1_4_zoom_flip.mp4
  123_2_4_zoom_flip.mp4
  123_3_4_zoom_flip.mp4
  208_1_2_zoom_edited.mp4

また、収集ミスにより、questionair.json には以下の対話のアンケートが欠落している。
